## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/your-username/NexusOS-AI.git
```

2. Navigate to the project folder:

```
cd NexusOS-AI
```

3. Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the main script:

```
python main.py
```
